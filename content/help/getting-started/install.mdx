title: 安装与唤醒扩展
description: 在 Chrome 或 Edge 安装智赢媒体助手，并用快捷键呼出侧边栏
---

## 安装扩展

1. 打开 [Chrome 网上应用店](https://chromewebstore.google.com/) 搜索「智赢媒体助手」，或从官网首页点击「安装扩展」。
2. 点击「添加至 Chrome」完成安装。
3. Edge 用户可在 Edge 加载项商店搜索同名扩展，流程相同。

## 唤醒侧边栏

1. 打开 [小红书网页版](https://www.xiaohongshu.com/)。
2. 按下快捷键 `Alt + C` 呼出全功能侧边栏。
3. 若首次使用，请确认浏览器已授予扩展读取当前页面内容的权限。

## 验证安装成功

侧边栏出现后，应能看到采集、导出、设置等入口。若未出现，请参阅 [侧边栏没有出现](/help/troubleshooting/sidebar-missing)。
