智赢媒体助手飞书同步功能可将采集数据一键传输至飞书多维表格，支持追加、覆盖、合并模式及图片上传。

遇到异常可前往 [飞书同步失败排查](/help/troubleshooting/feishu-sync-failed) 查阅解决方案。

## 获取飞书 App ID 和 App Secret

应用凭证配置步骤详见 [配置飞书自建应用](/help/getting-started/feishu-setup)，包括创建应用、导入权限、发布版本与复制凭证。

## 同步采集数据到飞书

数据采集完成后，在侧边栏点击**同步到飞书**（或对应模块的飞书同步按钮），即可打开同步配置弹窗。

飞书基础版每月 API 调用上限为 1 万次，其他飞书版本调用次数无限制。详见 [飞书官方说明](https://open.feishu.cn/document/home/user-guides/feishu-overview)。

### 表格链接

多维表格链接：打开需要同步的飞书多维表格，复制浏览器地址栏链接并粘贴到扩展中即可。

请确保配置的应用拥有该多维表格的**编辑权限**，否则将无法同步。可参考 [飞书同步失败排查](/help/troubleshooting/feishu-sync-failed) 为应用开启表格权限。

若扩展提供**创建**按钮，可自动创建多维表格并填入链接，无需手动复制。通过创建按钮生成的表格，权限通常为「互联网获得链接的人可编辑」。

### 同步模式

同步模式分为**追加**、**覆盖**和**合并**三种。无论哪种模式，都**不会删除现有字段**，最多只会删除记录。

**合并模式**

将新采集的数据与表格中的旧数据根据 ID 列进行合并：已存在的记录会覆盖，不存在的则新增。

> 存在多个重复值时，仅会覆盖第一个。

**追加模式**

将采集到的数据追加到表格末尾，不论数据是否已存在。

**覆盖模式**

删除表格中的现有数据，再将新采集的数据写入。

智赢媒体助手各采集模块可独立记住上次的目标表格，便于快捷一键同步。

### 上传素材

勾选后，可自动将表格中的图片、视频等媒体链接转换成文件同步到飞书表格中，例如：笔记封面、笔记图片、视频封面、评论图片等。

勾选后会明显增加同步耗时，且飞书对文件上传接口有限制，请按需开启。
